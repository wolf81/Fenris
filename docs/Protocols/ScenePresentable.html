ScenePresentable

public protocol ScenePresentable : SKScene

Scenes that conform to ScenePresentable are able to directly present other ScenePresentable scenes and able to dismiss themselves, though only when the container view is of type SceneView.

  • dismiss() Default implementation

    Dismiss this view, returning to the parent ScenePresentable view. Will not do anything if there is no parent. Please note: there is no real stack of views, instead views are recreated. It’s up to the developer to make sure the scene restores it’s proper state in the init(size:) constructor.
